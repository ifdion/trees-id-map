Map of Single Tree

This function turns an element into a leaflet map that display trees, based on its attributes. There are several method to query the trees :

  • by tree ids,
  • by tree offsets in conjuction with the lot id (similar with the single lot map with the tree offset attribute),

Usage

		<div id="trees-id-map" data-id="2550,2549"></div>

		<script type="text/javascript">
		treeMap('trees-id-map');
		</script>

Required Parameters / Attributes

data-id *
[string] Comma separated integer of tree ids to be displayed as marker
data-lot_id **
[int] The ID of the lot to be displayed
data-offset **
[int] tree offset to be displayed as marker
