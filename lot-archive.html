Lot Archive Map

This function turns an element into a leaflet map that display multiple lots, based on its custom attributes. Lot are queried by increment of 20 lot at a time. By default the map start as a heat map on smaller zoom level and will be converted into map of lot polygons on higher zoom level.

Usage

	<div id="trees-id-map" data-program_id="1" data-lot-page="http://v3.twitgreen.com/l/[lot]"></div>

	<script type="text/javascript">
		archiveMap('trees-id-map');
	</script>

Required Parameters / Attributes

data-lot-page
[string] Base URL template to be used as single lot URL.

Optional Parameters / Attributes

data-program_id
[int] Trees ID Program ids to be queried
data-project_id
[int] Trees ID Project ids to be queried
data-block_id
[int] Trees ID Block ids to be queried
data-relawan_id
[int] Trees ID Relawan ids to be queried
data-verifikator_id
[int] Trees ID Verifikator ids to be queried
data-petani_id
[int] Trees ID Petani ids to be queried
data-status_id
[int] Trees ID Lot Status to be queried
