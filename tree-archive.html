Map of Trees Archive

This function turns an element into a leaflet map that display multiple trees, based on its custom attributes. Trees are queried by increment of 20 tree at a time. By default the map start as a heat map on smaller zoom level and will be converted into map of tree markers on higher zoom level.

Usage

		<div id="trees-id-map" data-tree-page="http://portal.trees.id/tree-view-new/[lot]/[offset]" data-nohp="0811101272" ></div>

		<script type="text/javascript">
			archiveTreeMap('trees-id-map');
		</script>

Required Parameters / Attributes

data-tree-page
[string] Base URL template to be used as single tree URL. There are 3 special string available to construct the tree url. [id] will be replaced by the tree id, [lot] by lot id, [offset] by tree offset

Optional Parameters / Attributes

data-lot_id
[int] Trees ID Program ids to be queried
data-donatur_email
[string] Trees ID Project ids to be queried
data-code
[string] Trees ID Block ids to be queried
data-invoice
[string] Trees ID Relawan ids to be queried
data-nohp
[int] Trees ID Verifikator ids to be queried
data-affiliate
[string] Trees ID Petani ids to be queried
