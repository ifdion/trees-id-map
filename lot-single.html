Map of Single Lot

This function turns an element into a leaflet map that display a single lot. The map will be displayed as a zoomable heat map if the lot already has the tree photos data. As a fall back, the map will be displayed as a polygon if the data are not available. The heat map are responding to the tree cluster and will be displayed as marker on higher zoom level

Usage

		<div id="trees-id-map" data-id="248" data-tree-page="http://portal.trees.id/tree-view-new/[lot]/[offset]"></div>

		<script type="text/javascript">
		singleMap('trees-id-map');
		</script>

Required Parameters / Attributes

data-id
[int] The ID of the lot to be displayed
data-tree-page
[string] Base URL template to be used as single tree URL.

Optional Parameters / Attributes

data-offset
[string]Comma separated integer of tree offset to be displayed as marker
